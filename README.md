```

└── README.md
```

--app_database : Base de données importée.

#### Technologies utilisées

```
ReactJS pour le frontend
NodeJS pour les API REST
Sequelize ORM pour la bd MySQL 
```

#### Comment demarrer?

```
Ouvrir 3 fenetre de terminal

terminal 1:
    cd backend
    npm i
    npm start

terminal 2:
    cd frontend
    npm i
    npm start

 terminal 3:
    cd app_database
    npm i
    npm start
```
